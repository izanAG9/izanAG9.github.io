# Izan's Portfolio Website

This is the repository for my personal portfolio website, where I showcase my journey as a Cloud & Platform Support Engineer.

## Tech Stack
* **Framework**: [Hugo](https://gohugo.io/)
* **Theme**: [Toha](https://github.com/hugo-toha/toha)
* **Deployment**: GitHub Pages
* **Languages**: HTML, CSS, Markdown

## About the Project
This site serves as a live representation of my professional experience, technical skills, and ongoing projects. It is built using Hugo for performance and maintainability, deployed via GitHub Actions.
